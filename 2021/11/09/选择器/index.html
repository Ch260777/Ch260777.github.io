4、通配符选择器

表示选择所有元素

5、属性选择器

根据属性名和属性值选中元素
[href=”https://www.sina.com"]{
color: blue;
}
[href]{
color: blue;
}

6、伪类选择器

选中某些元素的某种状态
hover 鼠标悬停时的状态
:hover{
color:green;
}
active ：激活状态，鼠标按下时
:active{
color:rgb(45, 0, 128);
}
link 超链接未访问时的状态
visited 超链接访问过的
:link{
color:brown;
}
:visited{
color:burlywood;;
}

link>visited>hover>active

7、伪元素选择器

after
span::before{
content:”《”;

    }
    span::after{
        content:"》";
 
    }
